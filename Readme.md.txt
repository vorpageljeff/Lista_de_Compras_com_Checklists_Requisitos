# 🛒 Lista de Compras com Checklists

Aplicativo Flutter para criar e gerenciar listas de compras com interface simples, suporte a checklists e armazenamento local usando `shared_preferences`.

---

## ✨ Funcionalidades

- Adicionar itens à lista
- Marcar itens como comprados
- Armazenar dados localmente (funciona offline)
- Interface leve e responsiva

## ⚙️ Instalação e Execução

> Requisitos:

```bash
git clone https://github.com/vorpageljeff/Lista_de_Compras_com_Checklists_Requisitos.git
cd Lista_de_Compras_com_Checklists_Requisitos
